# R\&D





Puffpaw focuses on delivering services to early adopters who use verified devices during the initial three quarters. Access to the platform in this period is exclusively reserved for committed early supporters to assist in testing the mechanism.

Key features encompass:

* [x] Development and testing of hardware, including the structure and encrypted ID tracking system
* [x] Introduction of a beta interface featuring an in-dApp wallet, mobile App, and on-chain data processing capabilities
